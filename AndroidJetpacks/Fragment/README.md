# Fragment

```bash
├── 📁appbar
│   └── 📄 MenuProvider 를 사용하여 Fragment App Bar 만들기
├── 📁container
│   └── 🔥 FragmentContainerView 활용 예시
├── 📁createFragment
│   ├── 📁 baseFragment
│   │   ├── 📄 간단하게 Fragment 호스팅하는 방법
│   ├── 📁 fragmentFactory
│   │   ├── 🔥 FragmentFactory instantiate 구현 방법
│   ├── 📁 newInstance
│   │   ├── 🔥 AndroidX instantiate 구현 방법 
│   └── 📁 savedInstanceState
│       └── 📄 Programmatically 구현 방법 및 프래그먼트 상태 유지
├── 📁dialog
│   └── 🔥 DialogFragment 으로 커스텀한 알림창 만들기
├── 📁fragmentState
│   └── 📄 화면 회전으로 인한 프래그먼트 상태 유지
├── 📁fragmentTransaction
│   └── 📄 add(), replace() 사용으로 프래그먼트 생명주기 확인하기
├── 📁lifecycle
│   └── 📄 LifeCycle.STATED ENUM 과 LifeCycle 확인하기
├── 📁network
│   └── 📄 Activity <-> Fragemnt 통신 또는 Fragment <-> Fragment 통신하기
│       ├── 🔥 ViewModel 사용하기 (LiveData, Observer)
│       └── 🔥 Fragment Result API 사용하기 (setFragmentResultListener, setFragmentResult)



```
